## Development
1.  Clone from repository.
    ```
    $ git clone https://github.com/kitacode/sociotics
    $ cd sociotics
    ```
2.  Load development environment.
    ```
    $ . venv.sh
    ```
3.  Run test.
    ```
    $ python3.6 tests/test.py
    ```
